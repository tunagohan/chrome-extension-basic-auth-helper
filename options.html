Basic Auth Helper

id / password / 複数URL を登録して、 Basic認証を自動入力します。URLは1行に1つ入力してください。

ID
Password
URLs (1行に1URL)
Enabled
Action

Export / Import

JSON例:
[ { "id": "your-id", "password": "your-password", "urls": [ "https://work.andpaddev.xyz", "https://develop-work.andpaddev.xyz/" ], "enabled": true } ]
